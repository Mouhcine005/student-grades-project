# Student Data Cleaning Project

This project is a Python script that cleans student data from a CSV file. It performs tasks such as:
- Normalizing column names and text
- Handling missing values
- Converting values (e.g., male/female to m/f)
- Creating new columns (e.g., average grade)
- Saving the cleaned data to a new CSV file

## How to use

1. Place your student CSV file in the project folder.
2. Run the Python script:  
python main.py
The cleaned CSV file will be created in the same folder.

Requirements
Python 3.x

pandas

numpy

Install dependencies using:

pip install -r requirements.txt
